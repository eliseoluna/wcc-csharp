# C# Projects

A collection of C# coursework projects built for CIS285 using Windows Forms and .NET Framework 4.7.2.

This repository includes introductory desktop applications that focus on core programming skills such as event-driven UI development, input validation, arrays, game logic, and object-oriented programming.

## Featured Projects 

### Morse Code Generator

A Windows Forms app that converts plain text into International Morse code.

- Demonstrates input validation, text processing, and array-based lookup logic
- Project folder: `Project8/WindowsFormsApp1`

### Tic-Tac-Toe Simulator

A Windows Forms app that simulates a tic-tac-toe board, checks for winning combinations, and highlights results.

- Demonstrates 2D arrays, randomized board generation, and win-condition logic
- Project folder: `Project7`

### Production Worker OOP Demo

A Windows Forms app that captures employee data and demonstrates inheritance with custom C# classes.

- Demonstrates classes, properties, encapsulation, and inheritance
- Project folder: `Project10`

### Number Guessing Game

A Windows Forms game where the user guesses a random number and receives feedback after each attempt.

- Demonstrates random number generation, validation, and simple game-state management
- Project folder: `Project5`

## Tech Stack

- C#
- Windows Forms
- .NET Framework 4.7.2
- Visual Studio

## Running a Project

1. Open the desired project folder in Visual Studio on Windows.
2. Load the corresponding `.csproj` file.
3. Build and run the application.
