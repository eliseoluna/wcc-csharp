# Number Guessing Game

A C# Windows Forms game where the player guesses a randomly generated number from 1 to 100 and receives feedback after each attempt.

This project demonstrates input validation, random number generation, simple game-state tracking, and UI updates in a desktop application.

## Features

- Generates a random target number from 1 to 100
- Validates numeric user input
- Displays whether a guess is too high or too low
- Tracks the number of attempts
- Starts a new round after a correct guess

## Built With

- C#
- Windows Forms
- .NET Framework 4.7.2

## Concepts Demonstrated

- Random number generation
- Input validation with `int.TryParse`
- Conditional game logic
- Label and control state updates
- Event-driven programming in Windows Forms

## How to Run

1. Open the project in Visual Studio on Windows.
2. Load `Project5.csproj`.
3. Build and run the application.
